# JSON API Grails Reference

A reference implementation of [JSON API](http://jsonapi.org/) in [Grails 3](https://grails.org/).

Made for:
+ [JSON API 1.0](http://jsonapi.org/format/1.0/)
+ [Grails 3.3.0](http://docs.grails.org/3.3.0/)
